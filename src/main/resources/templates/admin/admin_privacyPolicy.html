【Do-It-Diet】プライバシーポリシー



このプライバシーポリシーは、[Do-It-Diet]（以下「当サービス」と称します）の利用者の個人情報の収集、利用、保護に関する方針を示しています。当サービスをご利用になる場合は、以下の内容をよくお読みいただき、同意していただく必要があります。

1. 収集する情報

当サービスでは、以下の情報を収集する場合があります:

・
個人を特定できる情報（氏名、連絡先情報など）
・
ユーザーのデバイス情報（IPアドレス、ブラウザの種類、利用環境など）
・
Cookieや類似技術を使用した情報

2. 収集目的

当サービスは、以下の目的で個人情報を収集します:

・
サービスの提供および運営
・
利用状況の分析および改善
・
問い合わせへの回答およびサポート提供
・
新規サービスや特典の案内

3. 情報の共有と第三者への提供

当サービスは、法的義務を遵守し、以下の場合を除き、ユーザーの個人情報を第三者と共有または提供しません:

・
ユーザーの同意がある場合
・
法的な要請がある場合
・
サービス提供のために必要な業務委託先と情報を共有する場合

4. 情報の保護とセキュリティ

当サービスは、ユーザーの個人情報を適切に保護するために技術的・組織的な安全対策を実施します。情報の漏洩、改ざん、不正アクセスなどに対するリスクに対して適切な対策を講じます。

5. ユーザーの権利

ユーザーは、自身の個人情報について以下の権利を行使することができます:

・
情報の確認、修正、削除の請求
・
個人情報の利用停止の請求
・
個人情報の第三者への提供停止の請求

6. プライバシーポリシーの変更

当サービスは、必要に応じてプライバシーポリシーを変更する場合があります。変更が行われた場合は、当サービスのウェブサイト上で通知いたします。
